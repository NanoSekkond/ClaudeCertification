# Claude Certification Implementations
## Como ejecutar sin problemas de dependencias
1. Crear un `.venv` con la extensión de Python en VSCode y activarlo con el ejecutable en la carpeta `bin`.
2. Correr `pip install -r requirements.txt` ya dentro del `.venv`.
3. Poner la API-Key en el `.env` con nombre `ANTHROPIC_API_KEY`.
4. Correr el siguiente comando: `python -m Path.Al.Archivo.nombre_del_archivo`.
